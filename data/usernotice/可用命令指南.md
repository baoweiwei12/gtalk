#### 可用命令指南

##### 删除指定用户

```GtalkShell
#gtalk delete [username]
```

##### 查询指定用户

```GtalkShell
#gtalk query [username]
```

##### 查询所有用户

```GtalkShell
#gtalk query-all 
```

##### 修改用户密码

```GtalkShell
#gtalk update-password [username] [password]
```

##### 修改用户有效期

```GtalkShell
#gtalk update-date [username] [xxxx-xx-xx]
```
